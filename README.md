# 📱 Student Registration App

A simple Android application developed using **Kotlin** in **Android Studio** that allows users to register student details and view them on a separate screen.

---

## 👨‍💻 Developed By
**Sheikh Wasimuddin**

---

## 📌 Project Description

This application demonstrates:

- Multi-Activity Navigation
- Form Input Handling
- Input Validation
- Data Passing using Intent
- Modern UI Design with CardView
- Use of Different Layout Types

---

## 🚀 Features

✔ 3 Activities (Screens)  
✔ Student Registration Form  
✔ Input Fields:
- Name  
- Email  
- Password  
- Phone Number  
- Roll Number  
- City  
- Gender (RadioButton)

✔ Input Validation:
- Empty field validation  
- Email format validation  
- Phone number length validation  
- Password length validation  
- Gender selection validation  

✔ Display Registered Data on Result Screen  
✔ Enhanced UI with CardView and Custom Colors  

---

## 🛠 Technologies Used

- Kotlin
- Android Studio
- ConstraintLayout
- LinearLayout
- CardView
- Intent (Data Passing)
- View Binding using findViewById()

---

## 📱 Application Flow

1️⃣ Main Screen  
→ Displays Developer Name and Register Button  

2️⃣ Registration Screen  
→ User enters student details  
→ Validation is applied  
→ Data is passed using Intent  

3️⃣ Result Screen  
→ Displays all entered details in a structured format  

---

## 📂 Layout Types Used

- ConstraintLayout  
- LinearLayout  
- ScrollView  
- CardView
